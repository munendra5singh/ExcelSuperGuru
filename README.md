# 📊 ExcelSuperGuru

**ExcelSuperGuru** is a modern, free, and interactive learning platform designed to help students, professionals, and job seekers master Microsoft Excel. From beginner concepts to advanced business analytics, the platform provides practical learning resources with a clean, responsive interface.

---

## 🚀 Features

* 📘 500+ Excel Formulas & Functions
* ⌨️ Keyboard Shortcuts (Windows & macOS)
* 📊 Interactive Excel Dashboard Tutorials
* 📈 Charts & Data Visualization
* 🧮 Real-Time Excel Practice
* 🎯 Interview Questions & Answers
* 📝 Excel Quizzes
* 📂 Downloadable Project Templates
* 💡 Excel Tips & Tricks
* 🌙 Dark & Light Theme Support
* 👥 Live Community Members (Firebase)
* 📱 Fully Responsive Design

## 🎯 Purpose

ExcelSuperGuru was created to provide a single platform where anyone can learn Microsoft Excel through practical examples, interactive tools, projects, and real-world business scenarios.

Whether you're preparing for interviews, improving office productivity, or building data analysis skills, this platform offers structured learning resources for every level.
