# 📊 ExcelSuperGuru

**ExcelSuperGuru** is a free and interactive Microsoft Excel learning platform built to help students, professionals, and data enthusiasts master Excel with practical examples.

## ✨ Features

* 📘 500+ Excel Formulas & Functions
* ⌨️ Keyboard Shortcuts
* 📊 Interactive Excel Tutorials
* 📈 Charts & Dashboards
* 🧮 Real-Time Excel Practice
* 🎯 Interview Questions
* 📝 Excel Quiz
* 💡 Tips & Tricks
* 🌙 Dark & Light Mode
* 📱 Fully Responsive Design
* 🔥 Live Community (Firebase)

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Firebase
* GitHub Pages

## 🎯 Mission

To provide a modern, free, and easy-to-understand platform where anyone can learn Microsoft Excel—from beginner to advanced—through interactive lessons, real-world projects, and practical exercises.
